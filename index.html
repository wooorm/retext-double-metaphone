retext-double-metaphone

A quick demo of retext-double-metaphone.

Just paste (or write) some text in the text area below, and next to it the HTML version of the document will appear, where every word is colour coded to its phonetics (words which sound the same are coloured the same).

The Double Metaphone algorithm is different from its predecessor in that it provides two possible pronunciations. In this example the secondary phonetics (if available) are visualised using a thick underline using the colour belonging to that phonetics.

Please raise any issues on Github.

Note! The generated colours are random (albeit generated from the phonetics): White colours might (will!) appear. Equal colours for different stems might also appear.
